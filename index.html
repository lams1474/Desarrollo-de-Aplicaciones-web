Semana 1: ¿Qué es el desarrollo de páginas web?

El desarrollo web abarca la creación, diseño y funcionamiento de sitios y aplicaciones accesibles desde un navegador. Tambien se introduce el modelo cliente-servidor, donde el navegador solicita información y el servidor la procesa y devuelve usando protocolos como HTTP y su versión segura HTTPS. También se presentan las áreas principales del desarrollo: frontend, backend y bases de datos.

Semana 2: Herramientas

Exploramos las herramientas esenciales para comenzar a desarrollar. Se utiliza Visual Studio Code como editor principal, un entorno que permite escribir codigo con resaltado, autocompletado y extenciones que amplian sus funciones. entre ellas destaca Live Server, que abre el proyecto en el navegador y actualiza los cambios en tiempo real. Tambien se revisan los navegadores como espacios de prueba y los servidores locales, que simulan el funcionamiento real de una pagina antes de publicarla.

Semana 3: HTML5 Semántico

Esta semana se profundiza en las nuevas etiquetas semánticas de HTML5, pensadas para dar estructura clara al contenido. Elementos como <header>, <section>, <article> y <footer> permiten que cada parte de la página tenga un propósito definido. Esto mejora la accesibilidad, ayuda a que los navegadores interpreten mejor el contenido y aporta beneficios directos al SEO, facilitando que buscadores comprendan la jerarquía de la información.
